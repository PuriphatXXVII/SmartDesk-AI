| Version | Supported |

## Reporting a Vulnerability

If you discover a security vulnerability in SmartDesk AI, please **do not** open a public issue.

Instead, email the maintainer at: **puriphat@example.com** (replace with your real address)

You can expect:
- Acknowledgement within 48 hours
- A fix or mitigation plan within 7 days for critical issues
- Public disclosure after a fix is shipped, with credit if desired

See [docs/SECURITY.md](docs/SECURITY.md) for our full security posture, threat model, and mitigations.
